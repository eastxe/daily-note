# daily-note

edit daily note @ ~/Documents/daily-note/

## Usage

```
note
```
open YYYYMMDD.md by vim

### option

```
note --atom
```
open file by atom

```
note [--name|-n] <filename>
```
open `<filename>.md`

## Install

```
git clone git@github.com:eastxe/daily-note.git
cd daily-note
python setup.py install
```
